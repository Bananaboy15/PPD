○이 프로그램에 대하여

  이 프로그램은 PPD 소프트웨어를 설치하는 프로그램입니다.
  또, 이 프로그램을 사용 하여 생기는 어떠한 문제에 대하여도 제작자는 책임을 지지 않습니다.

○사용법
PPDInstaller.exe실행한 뒤 설명에 따라 진행하십시요.

○지우기
프로그램은 수동으로 지우면 됩니다.

·PPD,PPDeditor,BMSTOPPD,DirectShowLib
　해당 폴더 및 파일을 지웁니다.

·IPA글꼴
　제어판에서 "IPAゴシック(IPAGothic)" 글꼴을 지웁니다.
　
·SlimDX,ffdshow
　제어판의 "프로그램 및 기능"에서 "프로그램 제거 및 변경"에서 지웁니다.
　
·MP4Splitter,FLVSplitter
　Windows+R 또는 시작-실행 에서 나온 창에
　
　regsvr32 -u MP4Splitter.ax (MP4Splitter를 지울때)
　regsvr32 -u FLVSplitter.ax (FLVSplitter를 지울때)
　
　을 입력한뒤 실행합니다. 그다음 Windows를 설치한 디스크(기본 C:\)
　
　C:\WINDOWS\system32\MP4Splitter.ax  (MP4Splitter를 지울때)
　C:\WINDOWS\system32\FLVSplitter.ax  (FLVSplitter를 지울때)
　
　위의 파일을 지웁니다.
