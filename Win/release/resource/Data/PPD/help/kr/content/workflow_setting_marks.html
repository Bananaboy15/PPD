이전까지해서 노트를 넣는것은 끝났습니다. 이번엔 노트의 위치, 각도, 종류를 설정해 봅니다.
일단 노트를 배치하는 방법에 대해 설명합니다.
직접 배치하려면 노트를 선택 후에 게임 창에서 끌어서 옮깁니다.


노트를 배치할 위치를 규칙적으로 하기 위해서 눈금을 사용할 수 있습니다.
눈금을 표시하면 가로 세로 눈금이 만나는 곳에만 노트를 배치할 수 있습니다.(눈금 단추로 바꿀 수 있습니다)


노트의 위치를 미세하게 조정할때는 방향 키(커서 키)를 사용합니다.(일반적으로 1px, +Shift 키로 10px, +Control 키로 5px씩 이동. 설정으로 바꿀 수 있습니다)
앞에 있던 같은 모양 노트의 위치를 복사하려면 Ctrl+Q를 누릅니다.
추가적으로위치 각도 로더 세이버-나지오메트리 크리에이터를 사용하면 배치 작업을 간단히 할 수도 있지만 이 도움말은 초보자용이므로 넘어가도록 합니다.

이번에는 노트의 각도입니다.
노트를 선택한 뒤 게임 창에서 노트 이외의 위치에서 끌면 각도를 조절할 수 있습니다.
이 때, Shift 키를 눌러 노트의 각도를 제한할 수 있습니다.(기본값은는 10도. 설정으로 바꿀 수 있습니다)


앞에 있던 같은 모양 노트의 위치를 복사하려면 Ctrl+R을 누릅니다.
추가적으로위치 각도 로더 세이버-나지오메트리 크리에이터를 사용하면 배치 작업을 간단히 할 수도 있지만 이 도움말은 초보자용이므로 넘어가도록 합니다.

다음은 노트의 종류에 대한 설명합니다.
노트를 넣는 가로 위치에 따라 노트의 종류가 바뀌는 것은 설명했습니다.
넣은 노트의 종류를 바꾸려면 Shift를 누르면서 노트를 끌어서 바꾸고 싶은 노트의 가로 위치에서 놓습니다.
Ctrl을 누르면서 끌어놓으면 노트가 복사됩니다.
다른 방법으로 여러 노트를 범위로 선택해서 Ctrl+C로 복사하면 노트를 넣을 위치에 Ctrl+V로 붙여넣을 수 있습니다.(Ctrl+Shift+C로 복사 버퍼를 비우는 것 잊지 마세요!)

롱노트(또는 HOLD)의 작성 방법입니다.
일단 롱노트 시작점(HOLD 시작점)에 일반 노트를 넣습니다.
그리고 롱노트 끝점(HOLD의 경우는 다음에 나올 노트 앞쪽에)에 노트를 하나 넣고 둘을 범위로 묶어 선택합니다.
이 상태에서 Ctrl+F를 누르면 롱노트(HOLD)로 바뀝니다.
롱노트를 선택한 뒤에 Ctrl+Shift+F를 누르면 일반노트로 나뉩니다
