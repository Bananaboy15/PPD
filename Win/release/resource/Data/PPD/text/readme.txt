○このソフトについて

  このソフトは、リズムゲームです。
  また、このソフトを使用して生じたいかなる損害も製作者は責任を負うことはありません

○動作環境

  WinXP,Vistaで確認済み
　また、必要なものとして

　.NET Framework 2.0 以降
　DirectX ランタイム
　SlimDX ライブラリ（SlimDX.dll)(2009 March SP1)
　Visual C++ 2008 ランタイムライブラリ
　DirectShow.NET 2.0　

　全てがインストールされている必要があります。
　また、MP4,FLVなどの動画を再生するためには
　ffdshowなどのDirectshowfilterをインストールし、かつ
　適切なスプリッターをインストールする必要があります

○使い方

　動画参照

○制作環境

    言語：C#,SlimDX
