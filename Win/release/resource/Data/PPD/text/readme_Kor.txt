○이 소프트에 대해

  이 소프트웨어는 리듬 게임입니다.
  이 소프트를 사용해 발생되는 어떠한 손해도 제작자는 책임을 지지 않습니다

○동작환경

  WinXP,Vista 실행 확인

　.NET Framework 2.0 이상
　DirectX 런타임
　SlimDX 라이브러리(SlimDX.dll)(2009 March SP1)
　Visual C++ 2008 런타임 라이브러리
　DirectShow.NET 2.0　

　모두가 설치되어야 합니다.
　MP4,FLV등의 동영상을 재생하기 위해서는
　ffdshow등의 Directshowfilter를 설치하고
　적당한 코덱을 설치해야 합니다

○사용법

　동영상 참조

○제작 환경

    언어：C#,SlimDX
